06 · Data

Sources & data

Everything that goes into the analysis is listed here, with its access status. If a source is paywalled or restricted, we say so.

Primary sources

These are the sources whose data directly feeds the calculations.

  • Data
    Open-Meteo Historical Weather API
    Hourly temperature, relative humidity, wind speed for Nouakchott (18.0858°N, −15.9785°W), 2019–2025. Free access, open API.
  • Curve
    ILO (2019), Working on a Warmer Planet
    ILO report on heat stress and worker productivity. Contains work capacity curves by intensity level. Free access (PDF on ilo.org).
  • Standard
    ISO 7243:2017
    International standard for heat stress assessment in the workplace. Defines WBGT thresholds by activity level. Paid access (iso.org), but curves are reproduced in the ILO report.
  • Stock
    JICA (2018), Nouakchott Urban Master Plan — Table I-14
    Inventory of Nouakchott's commercial spaces: 94,296 units. Most recent and comprehensive source on the city's commercial fabric. Free access (JICA report).
  • Tariff
    SOMELEC — Professional tariff schedule
    Rate of 5.903 MRU/kWh for professional consumers. Verified against SOMELEC's published tariff schedule.
  • Wage
    ILO ILOSTAT (2022) — Mauritania SMIG
    Minimum wage: 3,000 MRU/month. Used as the base unit for wage scenarios.

Reference sources

These contextualise the analysis but don't directly feed the calculations.

  • Context
    World Bank — Mauritania Profile
    Macroeconomic data, GDP, urban population. Used to contextualise orders of magnitude.
  • Context
    ONS Mauritania — Census and employment surveys
    Working population, urbanisation rate, employment structure. Used to validate the 120,000 exposed worker estimate.
  • Method
    Liljegren et al. (2008)
    Simplified formula for temperature/humidity → WBGT conversion. Published in the Journal of Occupational and Environmental Hygiene.

Reproducibility

All code and data are available on GitHub. You can reproduce every number in this analysis, modify assumptions, and test your own scenarios.

GitHub repository

github.com/agattbechar/nkc-heat-tax

Contains data download scripts, compression calculations, cost scenarios, and this site's source code.

If you find an error, a questionable assumption, or a better data source, open an issue on the repository. This work is open precisely so it can be challenged and improved.
