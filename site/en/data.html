Primary sources

Open-Meteo Historical Weather API

✓ Verified

Hourly temperature · 2019–2025 · lat 18.0858 · lon −15.9785 · UTC+0
License: CC BY 4.0 · Model: ERA5, Copernicus / ECMWF

Hourly temperature at 2 metres above ground for Nouakchott. 52,584 hours of data over the 2019–2025 period. Filtered to the 08:00–18:00 working window.

open-meteo.com/en/docs/historical-weather-api

ILO — Working on a Warmer Planet (2019)

✓ Verified

International Labour Organization · Geneva · ISBN 978-92-2-132968-9

Source of the work capacity curve by temperature. Moderate intensity: 200–300 W metabolic rate. Used for curve points and application methodology.

ilo.org — wcms_711919.pdf

ISO 7243:2017 — Heat Stress Standard

✓ Verified

International Organization for Standardization · Geneva · 2017

International occupational heat stress standard. Used alongside ILO (2019) to validate the work capacity curve thresholds.

iso.org/standard/67188.html

JICA — Nouakchott Urban Master Plan 2018

✓ Verified

Japan International Cooperation Agency · MHUAT Mauritania · Table I-14

Primary source for commercial stock estimation. Table I-14: land use composition — Commercial 952 ha · Mixed use 1,270 ha · Government 204 ha. Manually extracted from the final report.

openjicareport.jica.go.jp/pdf/12324729.pdf

SOMELEC — Tarifs Basse Tension

✓ Verified

Société Mauritanienne d'Électricité · Official tariff sheet

Tariff applied: INDUST/ARTISAN/COMMERCE, codes 7106–7136. Rate: 5.903 MRU/kWh. Used for AC premium calculation.

somelec.mr

ILO ILOSTAT — Mauritania Minimum Wage

✓ Verified

Dataset: EAR_INEE_CUR_NB_A · Mauritania · 2022 · MRU (new ouguiya)

Mauritanian SMIG: 3,000 MRU/month (2022). Used as lower bound for wage loss estimates. Note: verify if updated since 2022.

ilostat.ilo.org

OpenStreetMap — Commercial Buildings

✓ Verified

Overpass API · Tags: commercial, retail, shop, marketplace · Bbox: 17.95 / −16.08 / 18.18 / −15.89 · License: ODbL

168 features collected · 267,658 m² usable area (after filtering 7 outliers >20,000 m²). Used as lower bound reference only — OSM coverage estimated at ~15% of true stock.

overpass-api.de

Abandoned source

Google Maps Popular Times

⛔ Abandoned

Attempts: Maps UI · populartimes library · Places API · SerpAPI

Four attempts to obtain hourly behavioural data for Nouakchott — all unsuccessful. Cause: insufficient Google Maps user density in Nouakchott to generate Popular Times data. Replaced by the ILO/ISO curve — more defensible, fully citable, peer-reviewed.

Local knowledge

AC penetration rate — 1 in 3

~ Estimated

Source: local observation · Declared explicitly

Most Nouakchott commercial buildings have AC in the owner's office or back room — not full coverage. Rate used: 1 in 3. Three scenarios reported: 1/5 (conservative) · 1/3 (central) · 1/2 (upper). This assumption is declared explicitly in the methodology.
