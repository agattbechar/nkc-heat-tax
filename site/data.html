06 · Les données

Sources et données

Tout ce qui entre dans l'analyse est listé ici, avec son statut d'accès. Si une source est payante ou restreinte, nous le disons.

Sources primaires

Ce sont les sources dont les données alimentent directement les calculs.

  • Données
    Open-Meteo Historical Weather API
    Températures horaires, humidité relative, vitesse du vent pour Nouakchott (18.0858°N, −15.9785°W), 2019–2025. Accès libre, API gratuite.
  • Courbe
    ILO (2019), Working on a Warmer Planet
    Rapport de l'Organisation Internationale du Travail sur l'impact du stress thermique sur la productivité. Contient les courbes de capacité de travail par niveau d'intensité. Accès libre (PDF sur ilo.org).
  • Norme
    ISO 7243:2017
    Norme internationale pour l'évaluation du stress thermique en milieu de travail. Définit les seuils WBGT par niveau d'activité. Accès payant (iso.org), mais les courbes sont reproduites dans le rapport ILO.
  • Stock
    JICA (2018), Nouakchott Urban Master Plan — Table I-14
    Inventaire des espaces commerciaux de Nouakchott : 94 296 unités. Source la plus récente et la plus complète sur le tissu commercial de la ville. Accès libre (rapport JICA).
  • Tarif
    SOMELEC — Grille tarifaire professionnelle
    Tarif de 5.903 MRU/kWh pour les consommateurs professionnels. Vérifié sur la grille tarifaire publique de la SOMELEC.
  • Salaire
    ILO ILOSTAT (2022) — SMIG Mauritanie
    Salaire minimum : 3 000 MRU/mois. Utilisé comme unité de base pour les scénarios de salaire.

Sources de référence

Ces sources contextualisent l'analyse mais n'alimentent pas directement les calculs.

  • Contexte
    Banque Mondiale — Profil Mauritanie
    Données macroéconomiques, PIB, population urbaine. Utilisées pour contextualiser les ordres de grandeur.
  • Contexte
    ONS Mauritanie — Recensement et enquêtes emploi
    Population active, taux d'urbanisation, structure de l'emploi. Utilisées pour valider l'estimation de 120 000 travailleurs exposés.
  • Méthode
    Liljegren et al. (2008)
    Formule simplifiée de conversion température/humidité → WBGT. Publiée dans le Journal of Occupational and Environmental Hygiene.

Reproductibilité

L'intégralité du code et des données est disponible sur GitHub. Vous pouvez reproduire chaque chiffre de cette analyse, modifier les hypothèses, et tester vos propres scénarios.

Dépôt GitHub

github.com/agattbechar/nkc-heat-tax

Contient les scripts de téléchargement des données, les calculs de compression, les scénarios de coût, et le code source de ce site.

Si vous trouvez une erreur, une hypothèse discutable, ou une meilleure source de données, ouvrez une issue sur le dépôt. Ce travail est ouvert précisément pour être contesté et amélioré.
