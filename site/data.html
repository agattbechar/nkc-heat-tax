Sources primaires / Primary sources

Open-Meteo Historical Weather API

✓ Vérifié

Température horaire · 2019–2025 · lat 18,0858 · lon −15,9785 · UTC+0
Licence : CC BY 4.0 · Modèle : ERA5, Copernicus / ECMWF

Données horaires de température à 2 mètres du sol pour Nouakchott. 52 584 heures de données sur la période 2019–2025. Filtrées sur la fenêtre de travail 08h00–18h00.

open-meteo.com/en/docs/historical-weather-api

OIT — Working on a Warmer Planet (2019)

✓ Vérifié

Organisation Internationale du Travail · Genève · ISBN 978-92-2-132968-9

Source de la courbe de capacité de travail par température. Intensité modérée : 200–300 W de taux métabolique. Utilisé pour les points de la courbe et la méthodologie d'application.

ilo.org — wcms_711919.pdf

ISO 7243:2017 — Heat Stress Standard

✓ Vérifié

International Organization for Standardization · Genève · 2017

Norme internationale de stress thermique occupationnel. Utilisée conjointement avec l'OIT (2019) pour valider les seuils de la courbe de capacité de travail.

iso.org/standard/67188.html

JICA — Plan Directeur Urbain Nouakchott 2018

✓ Vérifié

Japan International Cooperation Agency · MHUAT Mauritanie · Table I-14

Source principale pour l'estimation du stock commercial. Table I-14 : composition de l'utilisation des terres — Commercial 952 ha · Usage mixte 1 270 ha · Gouvernement 204 ha. Extrait manuellement du rapport final.

openjicareport.jica.go.jp/pdf/12324729.pdf

Source abandonnée / Abandoned source

Google Maps Popular Times

⛔ Abandonné

Tentatives : Maps UI · populartimes library · Places API · SerpAPI

Quatre tentatives d'obtenir des données comportementales horaires pour Nouakchott — toutes infructueuses. Cause : densité insuffisante d'utilisateurs Google Maps à Nouakchott pour générer des données Popular Times. Remplacé par la courbe OIT/ISO — plus défendable, entièrement citable, peer-reviewed.

Connaissance locale / Local knowledge

Taux d'équipement AC — 1 sur 3

~ Estimé

Source : observation locale · Déclaré explicitement

La plupart des commerces de Nouakchott ont un climatiseur dans le bureau du propriétaire ou l'arrière-boutique — pas une couverture totale. Taux retenu : 1 sur 3. Trois scénarios reportés : 1/5 (conservateur) · 1/3 (central) · 1/2 (haut). Cette hypothèse est déclarée explicitement dans la méthodologie.
